Proyectos de Data Analyst

HR Analytics Dashboard

Desarrollo de un dashboard en Power BI para el análisis integral del talento, evaluando desempeño, engagement, capacitación y rotación de personal para apoyar la toma de decisiones en recursos humanos.

Insights clave

  • Distribución de desempeño concentrada en niveles medios, con oportunidad de impulsar alto rendimiento
  • Programas de capacitación interna más efectivos, aunque con mayor inversión
  • Engagement sin correlación clara con desempeño, sugiriendo influencia de otros factores
  • Proceso de reclutamiento con alta tasa de filtrado en etapas intermedias

Qué incluye el análisis

  • Evaluación de KPIs: headcount, rotación, desempeño promedio y tasa de selección
  • Análisis de desempeño por área y distribución de niveles de rendimiento
  • Evaluación de efectividad de capacitación (interna vs externa) y su impacto en desempeño
  • Análisis de engagement, satisfacción y balance vida-trabajo por empleado y área
  • Optimización del proceso de reclutamiento mediante análisis de embudo y tasa de conversión

Enfoque técnico

  • Modelado de datos relacional integrando múltiples fuentes (empleados, encuestas, capacitación y reclutamiento)
  • Diseño de modelo tipo estrella con relaciones uno a muchos para optimizar el análisis
  • Transformación y limpieza de datos en Power Query
  • Desarrollo de métricas en DAX para KPIs y análisis comparativo

Análisis de Proveedores y Compras (Supply Chain)

Desarrollo de dashboard en Power BI para evaluar el desempeño de proveedores, optimizar costos y monitorear indicadores clave en procesos de procurement.

Hallazgos clave

  • Alpha_Inc presenta el mejor desempeño general en cumplimiento y calidad, aunque no es el que genera mayor ahorro.
  • Existe un trade-off entre ahorro y desempeño operativo entre proveedores.
  • La categoría Office Supplies presenta la mayor tasa de defectos.
  • Las órdenes canceladas representan aproximadamente el 8% del total.

Qué incluye el análisis

  • Evaluación de KPIs: gasto total, ahorro, entregas y tiempos
  • Ranking de proveedores mediante el desarrollo de un modelo de evaluación para calcular un índice de desempeño
  • Análisis de calidad y defectos por categoría
  • Análisis temporal por año y cuatrimestre

Heart Failure Prediction

Desarrollo de modelo de Machine Learning para predecir la probabilidad de enfermedad cardíaca a partir de variables clínicas, con el objetivo de apoyar la detección temprana y la toma de decisiones médicas.

Qué hice

  • Limpieza y preprocesamiento de datos clínicos (edad, presión arterial, colesterol, entre otros)
  • Análisis exploratorio para identificar variables relevantes en la predicción
  • Entrenamiento y comparación de modelos de clasificación
  • Evaluación del modelo mediante métricas como accuracy, precision y recall

Resultados

  • Modelo capaz de identificar patrones asociados al riesgo de enfermedad cardíaca
  • Identificación de variables clínicas con mayor impacto en la predicción
  • Aplicación de técnicas de Machine Learning en un contexto de salud real

Tecnologías: Python | Pandas | Scikit-learn | Machine Learning
